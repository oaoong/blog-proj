# 서문

지난 7-9월 동안 ‘이펙티브 타입스크립트’ 도서에 대한 스터디를 진행했다.

개인적으로는 2회독이었는데, 혼자 공부한 1회독에서 책을 반도 못 흡수한 것 같아 아쉬움이 있었다.

스터디에서 함께 공부하고 발표하며 의견을 나누다보니 훨씬 쉽고 깊게 이해할 수 있어 좋았다.
